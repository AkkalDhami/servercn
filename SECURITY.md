# Security Policy

## Reporting a Vulnerability

If you discover a security issue in **ServerCN**, please report it privately.

- GitHub Security Advisories: <https://github.com/akkaldhami/servercn/security/advisories/new>

- Discord: <https://discord.gg/Cutb3vPG>

Please avoid public disclosure until the issue is resolved.

## Supported Versions

Only the latest version of ServerCN is actively supported with security updates.

## Responsible Disclosure

We appreciate responsible disclosure and will review reports as quickly as possible.
