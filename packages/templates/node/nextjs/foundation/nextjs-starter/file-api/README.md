# Next.js × Servercn

A Next.js starter template for Servercn, featuring Tailwind CSS, Shadcn, dark mode, and more.
---

## ⚡ Quick Start

```bash
npm install
npm run dev
```

---

## 🧠 Why Servercn?

Servercn turns Next.js into a **structured, backend-ready architecture**:

- API-first approach
- Scalable project structure
- Built for real-world applications

---

## 🚀 Extend This Starter

Upgrade anytime:

- MongoDB (Mongoose / Prisma)
- PostgreSQL (Drizzle / Prisma)
- MySQL (Drizzle / Prisma)

---

## ⚙️ Generate More

```bash
npx servercn-cli init nextjs-starter
```
