# TypeScript

TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale.

[TypeScript Official Website](https://www.typescriptlang.org/)

## Installation

Install the TypeScript configuration using the ServerCN CLI:

<PackageManagerTabs command="npx servercn add tooling typescript" />

## Configuration

The component provides a <Code children="tsconfig.json" /> file optimized for modern Node.js backend development.

<Code children="tsconfig.json" />

### 1. Minimal
```json
{
  "compilerOptions": {
    "target": "ES2021",
    "module": "es2022",
    "moduleResolution": "bundler",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "outDir": "dist",
    "sourceMap": true,
    "rootDir": "src",
    "alwaysStrict": true,
    "useUnknownInCatchVariables": true,
    "forceConsistentCasingInFileNames": true,
    "baseUrl": ".",
    "paths": {
      "@/*": [ "./*" ],
      "@/src/*": [ "./src/*" ]
    }
  },
  "include": [ "src/**/*" ],
  "exclude": [ "node_modules" ]
}
```

### 2. Advanced

```json
{
  "compilerOptions": {
    /* Base Options */
    "target": "ES2021",
    "module": "es2022",
    "moduleResolution": "bundler",
    "lib": ["ES2022"],
    "allowJs": true,
    "checkJs": false,
    "declaration": true,
    "sourceMap": true,
    "outDir": "dist",
    "rootDir": "src",
    "removeComments": true,

    /* Strictness */
    "strict": true,
    "noImplicitAny": true,
    "strictNullChecks": true,
    "strictFunctionTypes": true,
    "strictBindCallApply": true,
    "strictPropertyInitialization": true,
    "noImplicitThis": true,
    "useUnknownInCatchVariables": true,
    "alwaysStrict": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "noImplicitReturns": true,
    "noFallthroughCasesInSwitch": true,

    /* Module Resolution */
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    },
    "esModuleInterop": true,
    "forceConsistentCasingInFileNames": true,

    /* Advanced */
    "skipLibCheck": true,
    "resolveJsonModule": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist", "coverage", "**/*.test.ts"]
}
```

## Scripts

Add the following scripts to your <Code children="package.json" />:

```json
{
  "scripts": {
    "build": "tsc",
    "dev": "nodemon --watch ./src --exec tsx src/server.ts",
    "start": "node dist/server.js"
  }
}
```

## nodemon.json

<Code children="nodemon.json" />

```json
{
  "watch": ["src"],
  "ext": "ts,js,json",
  "ignore": ["node_modules"],
  "exec": "ts-node src/server.ts"
}
```

## Features

- **Strict Type Checking**: Enabled by default to catch bugs early.
- **Modern ES Support**: Pre-configured for the latest ECMAScript features.
- **Source Maps**: Configured for easier debugging.
- **Clean Output**: Separates source code from compiled JavaScript.
