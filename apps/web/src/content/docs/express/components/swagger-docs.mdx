# Swagger API Documentation

The **Swagger API Documentation** component provides an automated way to generate and serve interactive API documentation.

---

## Installation

Install the component using the servercn CLI:

<PackageManagerTabs command="npx servercn-cli add swagger-docs" />

---

## Implementation Guide
<Steps>
<Step>Configuration</Step>

The configuration file defines the metadata for your API and where Swagger should look for documentation comments.

```ts title="src/swagger.config.ts"
import swaggerAutoGen from "swagger-autogen";

const doc = {
  info: {
    title: "Stateless Auth API",
    description: "Stateless Auth API",
    version: "1.0.0"
  },
  host: "localhost:8000",
  schemes: ["http"]
};

const outputFile = "./docs/express/swagger.json"; // Output file for the generated docs
const endpointsFiles = ["./routes/*.ts"]; // Endpoints files to be parsed

swaggerAutoGen(outputFile, endpointsFiles, doc);
```

---

<Step>Run the Script</Step>

Run the script to generate the <Code children="swagger.json" /> file.

```bash
node src/swagger.config.ts
```

It will generate a <Code children="swagger.json" /> file in the <Code children="docs" /> directory.

---

<Step>Setup Swagger</Step>

Create a <Code children="swagger.ts" /> file in the <Code children="src/configs" /> directory.

```typescript title="src/configs/swagger.ts"
import swaggerUi from "swagger-ui-express";
import { Express } from "express";

import swaggerDocument from "../docs/express/swagger.json"; // Generated by swagger-autogen by running `node src/swagger.config.ts`

export const setupSwagger = (app: Express) => {
  app.use("/api/docs", swaggerUi.serve, swaggerUi.setup(swaggerDocument));
};
```

---

<Step>Integration with App</Step>

Update your main <Code children="app.ts" /> to initialize the Swagger documentation.

```ts {2, 7, 11} title="src/app.ts"
import express from "express";
import { setupSwagger } from "./configs/swagger";

const app = express();

// Initialize Swagger
setupSwagger(app);

app.listen(8000, () => {
  console.log("Server started on port 8000");
  console.log("Swagger docs available at http://localhost:8000/api/docs");
});
```

---

<Step>Update swagger.json file (Optional)</Step>

Update the <Code children="swagger.json" /> file with the new API documentation.

</Steps>

---

## Accessing the Docs

Once your server is running, navigate to:

<Code children="http://localhost:8000/api/docs" />

You will see the Swagger UI dashboard where you can explore and test your API.

<Warning text="Don't forget to run `node src/swagger.config.ts` command and update the base url, routes... in `swagger.json` file." />
